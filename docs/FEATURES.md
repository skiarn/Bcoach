# 🚀 Features (MVP)

## 1. Video Upload
- Ladda upp video från enhet
- Visa preview direkt

## 2. Video Player
- Play / Pause
- Slow motion (0.5x)
- Scrubbing

## 3. Drawing Tools
- Rita linjer
- Rita cirklar
- Radera

Syfte:
- Visualisera teknik (armvinkel, position)

## 4. Feedback Panel
- Fördefinierade tips
- Enkel logik baserat på user input

## 5. Save Analysis
- Spara video + anteckningar i localStorage

## 6. History
- Lista tidigare analyser
- Klicka för att öppna igen
